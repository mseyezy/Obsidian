# 了解扣子（Coze）

> 原始文件：[[raw/articles/了解扣子]]
> 在线链接：https://docs.coze.cn/

## 一句话摘要
[[Coze|扣子]] 是面向 Agent 时代的新一代 AI 团队协作平台：人和 [[Agent]] 像真正的团队一样共同工作，可一人调度多 Agent，也可多人协同多 Agent，在同一项目里完成办公、编程、视频创作、文件处理、行业研究等复杂任务，并持续沉淀资产。

## 核心要点
1. **多人多 Agent 协作**：支持一人+多 Agent、多人+多 Agent；不同 Agent 各司其职，共享上下文，@ 一下即可加入。
2. **多项目独立管理**：每个项目拥有独立对话、文件、资产，任务背景与产出持续沉淀。
3. **专业伙伴开箱即用**：Coze Agent 持续在线，支持 Doubao-Seed、Kimi、GLM、Minimax 等主流模型切换，结合长期记忆与技能调用。
4. **本地 Agent 接入**：可接入本地电脑上的 Claude Code、Codex CLI、OpenClaw 等 Agent，统一托管调度。
5. **Vibe Coding**：无需编程环境，手机也能让 Agent 拆解需求、生成代码、预览并部署。
6. **内容创作**：从长文、PPT 到全流程视频创作（剧本→分镜→生成→导出剪映工程）。
