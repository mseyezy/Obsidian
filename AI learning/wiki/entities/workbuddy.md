# WorkBuddy

**WorkBuddy** 是 [[tencent|腾讯]] 出品的 AI 办公工作台：用 [[natural-language-task-driving|自然语言任务驱动]] 直接完成办公任务，能读写本地文件、加载 [[skills-system|Skills 技能系统]]，并支持 [[ai-automation-pipeline|零代码自动化流水线]]。

## 六大核心功能（攻略中归纳）
1. 自然语言任务驱动
2. 本地文件操作（直接读写电脑文件）
3. Skills 技能系统
4. 社区
5. 积分体系（见 [[credits-points|积分 / 积分自由]]）
6. 实战场景演示

## 典型用法
- 新手：下载安装 → 核心功能上手 → 社区发文赚积分。
- 进阶：用自动化流水线做每日签到、批量整理文件、一键周报。

## 定位对比
与 [[coze|扣子 / Coze]] 同属"Agent 时代工作台"，但 WorkBuddy 更偏**个人办公提效与本地文件操作**，扣子更偏**多 Agent 团队协作与创作**。详见 [[workbuddy-vs-coze|WorkBuddy vs 扣子]]。
