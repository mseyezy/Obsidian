# 扣子 (Coze)

**扣子（Coze）** 是 [[bytedance|字节跳动]] 推出的面向 Agent 时代的新一代 AI 团队协作平台。人和 [[Agent]] 像真正的团队一样工作：一人可调度多 Agent，多人也可协同多 Agent，在同一项目里完成办公、编程、视频创作、文件处理、行业研究等任务，并持续沉淀资产。

## 核心能力
- **多 Agent 协作**：一人/多人 + 多 Agent，共享上下文，@ 一下即加入（见 [[multi-agent-collaboration|多 Agent 协作]]）。
- **多项目独立管理**：对话/文件/资产独立沉淀。
- **专业 Agent 开箱即用**：Doubao-Seed、Kimi、GLM、Minimax 等模型可切换，结合长期记忆与技能。
- **本地 Agent 接入**：Claude Code、Codex CLI、OpenClaw 一键接入。
- **Vibe Coding**：手机也能让 Agent 写代码、部署（见 [[vibe-coding|Vibe Coding]]）。
- **内容创作**：长文、PPT、全流程视频（剧本→分镜→生成→导出剪映工程）。

## 多端
网页端、桌面端（macOS/Windows，可远程操作本地文件）、移动端，跨端同步。

## 定位对比
与 [[workbuddy|WorkBuddy]] 同属 Agent 时代工作台，但扣子更偏**团队协作与创作**，详见 [[workbuddy-vs-coze|WorkBuddy vs 扣子]]。
