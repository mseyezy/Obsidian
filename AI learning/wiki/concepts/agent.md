# Agent (智能体)

**Agent（智能体）** 是可被赋予目标、自主调用工具/技能/记忆来完成任务的 AI 单元。在 Agent 时代，人和多个 Agent 像团队一样协作。

## 关键特征（以扣子 Coze Agent 为例）
- **持续在线**：长期记忆 + 技能调用 + 工作台能力，越用越懂你。
- **模型可切换**：Doubao-Seed、Kimi、GLM、Minimax 等主流模型自由切换。
- **专业可定制**：通过职业模板创建行业专家 Agent，或接入本地 Agent（Claude Code、Codex CLI、OpenClaw）。
