# 多 Agent 协作

**多 Agent 协作**指在 Agent 时代，人与多个 [[Agent]] 像团队一样共同工作：一人可调度多 Agent，多人也可协同多 Agent，共享上下文、分工推进同一项目。

## 在扣子（[[Coze]]）中的模式
- **一人 + 多 Agent**：自己调度多个专业 Agent 分工（资料整理、方案撰写、行业分析、代码、视频生成…）。
- **多人 + 多 Agent**：团队成员与 Agent 在同一项目共享上下文，@ 一下即可让某 Agent 加入。
- **资产沉淀**：每个项目拥有独立对话、文件、资产，成果持续复用。
- **本地 Agent 接入**：Claude Code、Codex CLI、OpenClaw 等本地 Agent 可一键接入，与云端 Agent 协同。

## 意义
把"单点对话工具"升级为"持续推进任务的协作工作台"，是 [[coze|扣子]] 与 [[workbuddy|WorkBuddy]] 这类新一代平台的核心差异点之一。
