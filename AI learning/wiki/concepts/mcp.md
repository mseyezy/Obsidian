# MCP（Model Context Protocol，模型上下文协议）

一种让 AI Agent 安全、标准化地接入外部工具与数据源的开放协议。通过 MCP，Agent 能调用数据库、API、本地应用等能力，突破"只能聊天"的边界。

## 作用
- **能力扩展**：把第三方服务（如各类 MCP 广场里的连接器）接进 Agent 工作流。
- **标准化集成**：一次接入，多处复用，降低定制成本。

## 在本文语境
