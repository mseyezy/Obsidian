# Skills 技能系统

**Skills 技能系统**是 [[WorkBuddy]] 等 AI 工作台的可扩展能力机制：把特定领域的知识/操作流程封装成"技能"模块，按需加载，让 Agent 获得专业执行力。

## 在 WorkBuddy 中的定位
- 属于其"六大核心功能"之一。
- 用户/平台可发布技能，社区共享，类似"应用商店"的能力插件。
- 与 [[natural-language-task-driving|自然语言任务驱动]] 配合：说一句话，调度对应技能完成。

## 类比
- 扣子（[[Coze]]）也有"行业专家技能一键加载"（自媒体/法律/金融/医疗等技能包）。
- 本质是同一思路：把可复用能力模块化、可发现、可组合。
