# Vibe Coding

**Vibe Coding** 是一种"用自然语言驱动编程"的范式：无需传统研发工具与编程环境，用户用一句话描述想法，AI 帮你拆解需求、生成代码、预览效果并部署。

## 在扣子（[[Coze]]）中的体现
- 支持网页端、桌面端、移动端多端数据同步，不在电脑旁也能搭建网站/App/小程序。
- 调用扣子编程（code.coze.cn）的 AI 编程能力：从想法→需求拆解→代码→预览→一键部署。
- 支持多人协作与多会话并行开发，可导入 GitHub / 本地项目继续开发。

## 与"自然语言任务驱动"的关系
都是"描述即执行"的交互范式，只是领域不同：Vibe Coding 面向**编程产物**，自然语言任务驱动面向**办公/文件任务**。
