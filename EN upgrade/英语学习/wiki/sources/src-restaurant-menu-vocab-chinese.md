# 中英文菜单词汇一（饮食英语词汇）

## 一句话摘要
一份日式料理餐厅菜单的中英对照词汇表，按料理类别组织，覆盖开胃菜、生鱼片、天妇罗、寿司、主食、面类、铁板烧、酒水等菜名。

## 核心要点
1. 按料理类别组织：开胃菜 → 生鱼片 → 烧烤 → 煮物 → 天妇罗 → 寿司 → 主食/面 → 串烧 → 铁板烧 → 酒水
2. 大量日料专有名词（sashimi, tempura, sushi, teppanyaki, teriyaki, shabu-shabu, sukiyaki）
3. 含套餐分类（set / teishoku）
