# 餐厅高频词发音要点

## 易错词发音（来自点餐指南）

| 词 | 音标 | 注意 |
|----|------|------|
| meal | /mil/ | 注意是 /i/ 长音，不是 /ɛ/ |
| dish | /dɪʃ/ | /ɪ/ 短音 |
| breakfast | /ˈbrɛkfəst/ | 重音在首 |
| entrée | /ˈɑntreɪ/ | 法语来源，元音是 /ɑ/ |
| main | /meɪn/ | 双元音 /eɪ/ |
| menu | /ˈmɛnju/ | 重音首，/ɛ/ 不是 /aɪ/ |
| gluten-free | /ˈglutən fri/ | 两词都轻读尾 |
| vegetarian | /ˌvɛʤəˈtɛriən/ | 重音在 ta（第三音） |
| vegan | /ˈviɡən/ | 简单两音节 |
| tip | /tɪp/ | 短音 |
| reservation | /ˌrɛzərˈveɪʃən/ | 重音在 va |

## 练习
- 跟读：Could I see the **menu**? /kəd aɪ si ðə ˈmɛnju?/
- 跟读：I'll have the **entrée**. /aɪl həv ði ˈɑntreɪ./
