# entree vs main course

⚠️ 这个词组最容易踩坑：entrée 的意思在美式/英式里是反的。

## 美式英语
- **entrée** = 主菜（= main course）
- The **entrée** comes with vegetables. 主菜配蔬菜。

## 英式/法式英语
- **entrée** = 开胃菜/正餐前的小菜（= starter）
- 法式套餐顺序：hors d'oeuvre → **entrée** → main → dessert

## main course（主菜）
全球通用，不会混淆。
- For my **main course**, I'll have the steak.

## 实用建议
- 在美国餐厅说 entrée，指主菜 ✓
- 在英/法餐厅说 entrée，指开胃菜 ⚠️
- 不想出错就统一用 **main course** 表主菜、**starter** 表开胃菜
