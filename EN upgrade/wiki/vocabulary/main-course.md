# main course

## 释义
主菜（一顿饭的主菜部分）。

## 例句
- For my **main course**, I'll have the steak. 主菜我点牛排。

## 搭配
- the main course（主菜）
- starter → main course → dessert（开胃菜→主菜→甜点）

## 近义/反义
- 美式常称 [[entree]]
- 对比：[[starter]]（开胃菜）、dessert（甜点）
