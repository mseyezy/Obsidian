# 如何用英语点餐？100+种表达助你享受美食

## 一句话摘要
实用指南：从预订餐桌、堂食点餐、电话订餐到结账，覆盖餐厅场景 100+ 英语表达，含音标与发音。

## 核心要点
1. 点餐场所分类：restaurant / café / takeout / food truck
2. 餐厅核心词汇表（含音标）：meal, dish, menu, starter, entrée, main, knife, fork, chopsticks...
3. 预订餐桌句型：make a reservation / book a table
4. 堂食点餐句型：Could I see the menu? / I'll have the... / May I have the check?
5. 电话点餐对话样本
6. 菜单食物词汇（含甜点）
7. 3 个本地人技巧：礼貌、给小费、标价不含税
